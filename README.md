
## Cub3D

My first RayCaster with miniLibX

```
To Run the Game you need Minilibx library, C compiler and Makefile installed.
```

```
git clone https://github.com/sumon-ohid/42_Cub3D.git
cd 42_Cub3D
make
./cub3D maps/simple.cub
```
